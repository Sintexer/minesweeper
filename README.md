# Minesweeper

A classic Minesweeper game made with Godot 4 featuring responsive UI.

**Features:**
- Several difficulty levels
- Juicy clicks
- Responsive UI

 ## How to play

Deployed on itch.io (paste link here)

Reveal all safe tiles without hitting a mine. Numbered tiles show how many mines are around this tile (3x3 box). Use flags to mark suspicious tiles and prevent accidental reveal.

Local build is available using godot 4
